# 🍽️ MentraResto Lite - كاشير ونظام إدارة المطاعم (Offline-First)

![Version](https://img.shields.io/badge/version-1.0.0--Lite-orange?style=for-the-badge)
![Platform](https://img.shields.io/badge/platform-Web%20%7C%20PWA%20%7C%20Desktop-blue?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-green?style=for-the-badge)

**MentraResto Lite** هو نظام كاشير ونقاط بيع (POS) متطور ومبتكر، مصمم خصيصاً لخدمة المطاعم والكافيهات بكفاءة عالية وبمنهجية **Offline-First**. يعمل النظام بالكامل داخل المتصفح أو كتطبيق مستقل دون الحاجة إلى اتصال نشط بالإنترنت أو الاعتماد على خوادم سحابية خارجية، مما يضمن أقصى درجات السرعة، الاستقرار، وخصوصية البيانات.

---

## ✨ المميزات الرئيسية (Key Features)

*   ⚡ **استجابة لحظية وسرعة فائقة:** لا وجود لزمن تحميل الخوادم، يتم فتح وإغلاق الفواتير وتنفيذ العمليات بشكل فوري بفضل الاعتماد الكامل على قاعدة البيانات المحلية.
*   🔒 **أمان وخصوصية تامة:** بيانات المبيعات، الفواتير، والعملاء تُخزن محلياً على جهازك بنسبة 100%، وهي ملكك وحدك ولا يمكن لأي طرف ثالث الوصول إليها.
*   🛒 **كاشير ونقاط بيع متكاملة:** إدارة احترافية لطلبات الصالة (الطاولات)، التيك أواي، وطلبات الدليفري مع دعم مرن لإضافة الضرائب والخصومات.
*   💰 **إدارة الخزينة والحسابات:** مراقبة دقيقة للإيرادات والمصروفات اليومية، وحساب صافي الأرباح بضغطة زر واحدة.
*   🏢 **دعم تعدد المطاعم والفروع:** إمكانية تشغيل وإدارة أكثر من مطعم أو فرع مستقل تماماً بقواعد بيانات منفصلة على نفس الجهاز.
*   📱 **تطبيق ويب تقدمي (PWA):** يدعم التثبيت المباشر على الهواتف والأجهزة اللوحية كأنه تطبيق محلي مدمج، ويعمل بدون إنترنت تماماً.
*   💻 **نسخة مكتبية مستقلة:** توفر نسخة جاهزة للعمل على أجهزة الكمبيوتر (Windows) لتجربة كاشير تقليدية ومستقرة.

---

## 🚀 التقنيات المستخدمة (Tech Stack)

تم بناء النظام باستخدام تقنيات خفيفة وحديثة لضمان أفضل أداء واستهلاك أقل للموارد:

*   **Front-End:** HTML5, Tailwind CSS (عبر الـ CDN لسهولة التشغيل والتخصيص الرسومي)، FontAwesome 6.
*   **Local Storage & Database:** `IndexedDB` (عبر المتصفح) لإدارة وتخزين البيانات الضخمة والمعقدة محلياً بسلاسة وسرعة.
*   **PWA Core:** Service Workers لعمل الكاش وتأمين عمل النظام دون إنترنت، وملف `manifest.json` لتهيئة التثبيت.
